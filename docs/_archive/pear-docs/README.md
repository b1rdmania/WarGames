# Pear Protocol docs (Firecrawl extracts)

These markdown files are **Firecrawl extracts** of the Pear Protocol docs pages we actively implement against.

- They’re intended to be **versioned references** for hackathon development and debugging.
- When changing integration code, reference the relevant file(s) below in commit/PR notes.

## Extracts

- `AUTHENTICATION.md` — `GET /auth/eip712-message`, `POST /auth/login`, `POST /auth/refresh`
- `AGENT_WALLET.md` — `GET /agentWallet`, `POST /agentWallet`
- `VAULT_WALLET.md` — `GET /vault-wallet/balances`, transfers, vault wallet endpoints
- `POSITIONS.md` — `GET /positions`, `POST /positions`, `POST /positions/{id}/close` (and more)
- `WEBSOCKET.md` — websocket endpoint + channel subscription format
